Contains LVG general library classes of BitMaskBase, Category, Gender, Inflection, Configuration, CombineRecords, OutputFilter, Flow, GlobalBehavior, LexItem, LexItemComparator. This package serves as a library for Lexical Tools and is not intended to be used by end users directly. End users may skip this package.

This library can be categorized as follows:

  • BitMask Operations:
    • BitMaskBase
    • Category
    • Gender
    • Inflection
    • Flow

  • System Options:
    • Configuration
    • CombineRecords
    • OutputFilter
    • GlobalBehavior

  • LexItem Related:
    • LexItem
    • LexItemComparator
